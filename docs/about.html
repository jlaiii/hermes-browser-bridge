About

hermes-browser-bridge is a lightweight, CSP-safe browser automation tool built for the Hermes Agent ecosystem. It lets you remotely control live browser tabs via a local relay server, using Tampermonkey's privileged GM_xmlhttpRequest to bypass strict Content-Security-Policy rules on sites like GitHub, LinkedIn, ChatGPT, and eBay.

Who Built It

Built by jlaiii for the Hermes Agent project — an AI-powered CLI automation environment running inside WSL.

Why It Exists

Traditional browser automation tools (WebSocket injections, Puppeteer, Playwright) fail on heavily CSP-protected sites. This bridge uses HTTP short-polling, so it works everywhere Tampermonkey does.
