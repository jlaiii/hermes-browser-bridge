Features

Everything a browser automation bridge needs to work in the real world.

CSP Safe

WebSocket-based tools fail on GitHub, LinkedIn, and Twitter. We use HTTP short-polling via GM_xmlhttpRequest — no CSP can block it.

Tampermonkey Native

Runs as a userscript — auto-installs on every page. No browser extensions to package. No store approval waits.

Local Relay

Tiny aiohttp relay server on localhost:8765. No cloud dependency. No subscriptions. Your browser stays yours.

Site Agnostic

Tested on ChatGPT, GitHub, LinkedIn, eBay, Twitter/X. If Tampermonkey runs on it, the bridge controls it.

Auto-Reconnect

Refresh a tab? Bridge turns green again in under 2 seconds. Move between pages? It follows. Network blip? It retries.

Multi-Tab Control

Every tab becomes its own client. Send a message to ChatGPT while scraping eBay — all from one relay.

Quick Start

From zero to controlling a live tab in under 60 seconds.

  1. Install Tampermonkey on Firefox, Chrome, or Edge.
  2. Click here to install the userscript. A panel appears on every page.
  3. Download and start the relay:
$ git clone https://github.com/jlaiii/hermes-browser-bridge.git $ cd hermes-browser-bridge $ pip install aiohttp && python hermes-browser-relay.py

Open any website. Watch the panel turn green. You're connected. See the full install guide.
