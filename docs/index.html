Automate the browser without breaking CSP

A lightweight, CSP-safe browser automation bridge. Use HTTP short-polling via Tampermonkey's GM_xmlhttpRequest and a localhost relay to control pages like ChatGPT, GitHub, LinkedIn, and eBay.

Get Started View on GitHub

Features

CSP-Safe

Works inside strict Content-Security-Policy environments where traditional injection fails.

Tampermonkey Native

Leverages GM_xmlhttpRequest for reliable cross-origin outbound requests.

Localhost Relay

A tiny local relay server bridges your automation scripts to the browser.

Site Agnostic

Supports ChatGPT, GitHub, LinkedIn, eBay, and any other site with a Tampermonkey context.

Short-Polling

Simple HTTP short-polling design; no WebSockets, no extra ports, no headaches.

Zero External Deps

The userscript and relay are self-contained with no third-party dependencies.

Quick Install

  1. Install the relay — clone the repo and start the relay server:
    git clone https://github.com/jlaiii/hermes-browser-bridge.git
    cd hermes-browser-bridge && python relay.py
  2. Install the userscript — copy the script into Tampermonkey (or Greasemonkey). It runs on target sites and polls http://localhost:8080 by default.
  3. Automate — send commands to the relay via HTTP; the userscript executes them in the browser and returns results.

See the full install guide for platform-specific copy-paste commands.

Architecture

+---------------------+ HTTP POST/GET +---------------------+ | Your automation | <------------------------> | localhost relay | | script / CLI | | (e.g. port 8080) | +---------------------+ +----------+----------+ | | fetch | +--------v---------+ | Target page | | (Tampermonkey) | | GM_xmlhttpRequest | +--------+---------+ | +--------v---------+ | Browser DOM | | Click / Type / | | Read / Navigate | +------------------+

Command Reference

Send these JSON payloads to the relay. All responses are returned via the short-poll loop.

CommandPayloadDescription
click{"cmd":"click","selector":"#btn"}Click a DOM element.
type{"cmd":"type","selector":"#input","text":"hello"}Type text into an input.
read{"cmd":"read","selector":".content"}Read innerText from an element.
navigate{"cmd":"navigate","url":"/page"}Change window.location.
eval{"cmd":"eval","js":"document.title"}Run arbitrary JS in page context.
