Install

Two pieces: a relay server on your machine, and a userscript in your browser.

1

Clone & Install

Clone the repo and start the relay. Python 3.8+ required.

git clone https://github.com/jlaiii/hermes-browser-bridge.git
cd hermes-browser-bridge
python3 -m venv venv
source venv/bin/activate
pip install aiohttp
python3 hermes-browser-relay.py
ℹ️

The relay listens on http://0.0.0.0:8765. Leave this terminal open while you use the bridge.

Download & Install

Download the repo, set up a venv, install deps, and run the relay.

git clone https://github.com/jlaiii/hermes-browser-bridge.git
cd hermes-browser-bridge
python -m venv venv
venv\Scripts\activate
pip install aiohttp
python hermes-browser-relay.py
ℹ️

If Python is not installed, get it from the Windows Store or python.org.

Verify It Works

Open any page in your browser. Look for the Hermes Bridge panel in the bottom-right.

curl -s http://localhost:8765/api/ping
# {"ok": true, "ts": 1234567890.0}
curl -s http://localhost:8765/api/clients
# {"clients": [{"id": "...", "url": "https://chatgpt.com"}]}
Copy

Green status = connected. Start sending commands!
