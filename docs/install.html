Install hermes-browser-bridge

Two parts: the userscript (browser) and the relay (localhost). Pick your OS below for exact copy-paste commands.

1. Clone the repository

All platforms use the same repo:

git clone https://github.com/jlaiii/hermes-browser-bridge.git
cd hermes-browser-bridge

2. Start the relay

Choose your platform to see the right command.

Linux macOS Windows
# Ensure Python 3 is installed, then start the relay
python3 relay.py
# macOS ships with Python 3 since Catalina; start directly
python3 relay.py
# In Command Prompt or PowerShell, run the relay
python relay.py

3. Install the Tampermonkey userscript

The userscript polls the relay and executes commands in the browser.

  1. Install Tampermonkey for your browser.
  2. Open Tampermonkey Dashboard → Utilities → Import from file.
  3. Select hermes-browser-bridge.user.js (inside the cloned repo).
  4. Save and enable the script for the sites you want to automate.

Alternatively, create a new script and paste the contents of hermes-browser-bridge.user.js directly.

4. Verify it works

Send a test ping and read the page title:

curl -X POST http://localhost:8080/api/command \
-H "Content-Type: application/json" \
-d '{"cmd":"eval","js":"document.title"}'

You should see the current page title returned in JSON.
